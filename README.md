# 🚀 Sorting Algorithms in Java

This repository contains Java implementations of fundamental **Sorting Algorithms**. The project is designed to strengthen the understanding of sorting techniques, their working principles, and their time and space complexities. These algorithms are essential for Data Structures & Algorithms (DSA) and technical interview preparation.

## ▶️ How to Run

```bash
git clone https://github.com/PalagiriNagayogeswari/Sorting.git
```

### Navigate to the project

```bash
cd Sorting
```

### Compile the Java file

```bash
javac FileName.java
```

### Run the program

```bash
java FileName
```

Replace `FileName` with the name of the sorting algorithm you want to execute.

## 📊 Time Complexity Comparison

| Algorithm | Best Case | Average Case | Worst Case | Space Complexity |
|-----------|-----------|--------------|------------|------------------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) |
| Counting Sort | O(n + k) | O(n + k) | O(n + k) | O(k) |

> **Note:** `k` represents the range of input values in Counting Sort.
